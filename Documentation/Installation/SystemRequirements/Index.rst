===================
System requirements
===================

:Operating System:
    Linux, Microsoft Windows, macOS (also when these are hosted on all common cloud environments)

:Webserver:  Apache httpd, Nginx, Microsoft IIS, Caddy Server
:Supported Browsers: Chrome,
                     Edge,
                     Firefox,
                     Safari
:Composer: Composer >= 2.1
:Database:
             MariaDB >= 10.4.3,
             MySQL >= 8.0.17,
             PostgreSQL >= 10.0,
             SQLite >= 3.8.3
:Hardware: RAM >= 256 MB
:PHP: PHP >= 8.2.0 <= 8.4.99

Further information about TYPO3 system requirements can be found at `get.typo3.org
<https://get.typo3.org/version/#system-requirements>`_.
