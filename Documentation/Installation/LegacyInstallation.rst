===================
Legacy Installation
===================

..  warning::
    TYPO3 can be installed without using Composer. This method of installation
    is now considered **out of date**, users are strongly encouraged to use the
    :ref:`Composer-based installation instructions <install>`.

See how to do a :ref:`Legacy installation in TYPO3 explained <t3coreapi:legacyinstallation>`.
