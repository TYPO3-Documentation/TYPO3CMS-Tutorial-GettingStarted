===========
For Editors
===========

Tutorial for Editors
====================

The :doc:`t3editors:Index` explains how to add and amend content on an existing TYPO3 based website.
It provides simple, step-by-step instructions that allows you to being using TYPO3 quickly.

Video
-----

.. youtube:: nl0mY6S1Pdg


Link: `Introduction to TYPO3 8 LTS for Editors <https://www.youtube.com/watch?v=nl0mY6S1Pdg>`__

.. important::

   This video is for TYPO3 8, but you will find that most of what is covered also
   applies to TYPO3 9.

You may also want to check out the
`TYPO3 YouTube channel <https://www.youtube.com/user/typo3/videos>`__
for more videos on TYPO3.
